# Flow Lounge

A modern web application for Flow Lounge, a premium hookah bar and lounge in Thessaloniki, Greece.

## Overview

Flow Lounge offers an immersive experience with premium hookah, craft cocktails, and a gaming lounge. This website showcases the menu, events, and services available at the establishment.

## Features

- Interactive menu with hookah, cocktails, drinks, and food options
- Events calendar and special promotions
- Gaming lounge (Playroom) information
- Smooth animations and modern UI
- Mobile-responsive design
- Multi-language support (Greek/English)

## Tech Stack

- Next.js 15.5
- React 19
- TypeScript
- Tailwind CSS 4
- GSAP animations
- Framer Motion
- Lenis smooth scroll

## About Flow Lounge

Flow Lounge is located in Thessaloniki, Central Macedonia, Greece. We offer over 200 unique hookah flavors, signature cocktails, and a premium gaming experience with PlayStation 5 and board games.
